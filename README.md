# PersonalSite

<img src="/public/cover.jpg" alt="Cover image representing my personal website" width="100%" />

This is my personal website built with Next.js 15, React 19, Tailwind CSS v4, and Motion. It features a clean, minimal design with delightful animated components powered by [Motion-Primitives](https://motion-primitives.com).

## Features

- Minimal one-page portfolio layout
- Blog support with MDX
- Responsive and accessible design
- Animated components using [Motion-Primitives](https://motion-primitives.com)

## Local Development

```bash
git clone https://github.com/LawrenceCirillo/PersonalSite.git
cd PersonalSite
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This site is deployed using Vercel.
